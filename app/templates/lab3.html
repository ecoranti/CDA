Generador de Curvas BER

Ejecuta una simulación barriendo Eb/N0 para obtener la curva BER, encadenada con la salida real de Modulación.

Cómo usar este bloque: si quieres una curva BER, define un rango con Eb/N0 mínimo, Eb/N0 máximo y Δ Eb/N0. Si quieres un punto de operación, fija Eb/N0 mínimo = Eb/N0 máximo; el sistema calculará un único valor de P_b = BER para ese Eb/N0.
Si está activo, la curva usa `iq.bin/iq_tx.bin` + bits de una corrida real de Modulación.
Si la ruta contiene subcarpetas (audio/texto y métodos), se procesan todas automáticamente.
Recomendado: 20 a 50 para estabilizar la curva BER.
