Recorrido del demodulador

1. Entrada {% if summary.channel_mode == 'ideal' %}
La señal IQ transmitida entra al receptor a través de un canal ideal sin ruido.
r[n] = s[n]
{% else %}
La señal IQ transmitida entra al receptor y se contamina con AWGN.
r[n] = s[n] + w[n]
{% endif %}
2. Filtro acoplado
Se aplica el RRC del receptor para maximizar SNR y controlar ISI.
z[n] = r[n] * h_MF[n]
3. Muestreo
Se toma una muestra por símbolo en los instantes óptimos.
z[kT_s]
4. Decisión
Las muestras se comparan con el umbral y se reconstruyen bits.
bits_rx
